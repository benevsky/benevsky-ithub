docxtpl + WeasyPrint
Шаблон .docx с Jinja2-переменными → auto-fill из Deal Entity + DealScoring. PDF рендер через WeasyPrint или LibreOffice headless. Watermark «Confidential» + NDA-disclaimer вшивается автоматически.
AI Executive Summary
GPT-4o генерирует нарративную часть (секции 2, 9, 10) на основе данных Q1/Q2 + скоринга. Промпт инжинирится под методологию AP. Аналитик может отредактировать перед выгрузкой.
Версионирование
Каждая генерация — новая версия (v1, v2…). Scoring snapshot сохраняется в JSONB. История доступна аналитику. Статусы: draft → approved → sent. S3 URL фиксируется в карточке Bitrix24.

React/Next.js PWA: авторизация через OTP, дашборд портфеля, deal flow с тизерами, P&L трекинг. Доступ к контенту разграничен по уровню Gold/Silver/Bronze/Observer.

Карта страниц инвесторского портала — UX Architecture
Auth: OTP + OAuth
Passwordless auth: email → OTP (6 digits, TTL 10 min) через SendGrid. Альтернатива: OAuth 2.0 (Google/Apple). JWT access token + refresh token rotation. MFA для операций с документами.
